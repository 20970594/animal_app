Problemática general:
Crear una app mediante flutter la cual pueda utilizar una API para obtener, enviar y compartir información.

Información de la App:
La app busca utilizar distintas API para mostrarle al usuario imágenes de Perros, Gatos y Patos.
Aparte, el usuario puede guardar imágenes en la APP o descargarlas en su celular.
Aparte puede actualizar para que le aparezcan fotos de otras mascotas.
