Nombre de la app: FGC Fighting Game Coach
Descripción: FGC es una aplicación para telefonos dirijida a los aficionados a los viodejuegos de lucha,
especialmente para aquellos que se toman la competitividad de forma mas seria y que buscan
mejorar de forma significativa. Esta aplicación permite al usuario mejorar a traves de un seguimiento
de rutinas para distintos juegos, cada uno con distintos niveles de dificultad.

Link al repositorio: https://github.com/20970594/FGC-Fighting_Game_Coach/tree/new_develop

•	Requerimientos funcionales:
o	El sistema debe permitir al usuario cambiar de videojuego.
o	El sistema debe permitir al usuario editar y visualizar rutinas de los juegos.
o	El sistema debe permitir al usuario ver su progreso de rutinas completadas y en progreso.

•	Requerimientos no funcionales:
o	El sistema debe tener una interfaz de fácil uso y navegación.
o	El sistema debe permitir una transición rápida entre páginas y videojuegos.
o	El sistema debe estar siempre disponible.
